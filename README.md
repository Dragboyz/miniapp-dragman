## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Dragboyz/miniapp-dragman.git
cd miniapp-dragman

# Install dependencies
npm install

# Run development server
npm run dev
```

### Production Deployment

```bash
# Build for production
npm run build

# Start production server
npm start
```
